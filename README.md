# ITパスポート 過去問トレーニング

IPAが公開しているITパスポート試験の過去問をブラウザで学習できるツール。
スマホ対応済み。出先でもサクッと勉強できます。

**🌐 https://ramdamain-commits.github.io/it-passport-training/**

## 使い方

上のURLをスマホ・PCのブラウザで開くだけ。インストール不要。

### モード選択

| モード | 説明 |
|--------|------|
| **学習モード** | 年度・分野を絞って自由に問題を解く。即時正誤フィードバック。シャッフル可 |
| **模擬試験** | 全年度から100問ランダム出題。120分タイマー。終了時に合格判定 |
| **苦手問題復習** | 模擬試験で2回以上不正解だった問題だけを抽出して出題 |
| **成績** | 模擬試験の得点推移・分野別正答率・苦手問題リストを確認 |

### 模擬試験の仕様

- 出題: ストラテジ35問 / マネジメント20問 / テクノロジ45問（実試験と同じ比率）
- 類似問題回避: 同じ問題番号は最大2年度まで
- 回答中は正誤を表示しない（本番同様）
- 合格基準: 総合60%以上 かつ 各分野30%以上（正答率ベースの簡易判定）
- 結果画面で全問の正誤一覧・詳細を確認可能

### キーボードショートカット（PC向け）

| キー | 動作 |
|------|------|
| `→` / `n` | 次の問題 |
| `←` / `p` | 前の問題 |
| `1` `2` `3` `4` | ア / イ / ウ / エ を選択 |

## 機能

| 機能 | 説明 |
|------|------|
| **学習進捗の自動保存** | 回答状況・正誤・フィルター状態を localStorage に保存。ブラウザを閉じても続きから再開できる |
| **成績画面** | 模擬試験の受験履歴を記録。直近10回の得点推移を SVG 折れ線グラフで表示 |
| **分野別正答率** | ストラテジ・マネジメント・テクノロジの正答率を横棒グラフで可視化 |
| **苦手問題リスト** | 模擬試験で2回以上不正解だった問題を一覧表示 |
| **苦手問題復習モード** | 苦手問題リストの問題だけを抽出して出題。弱点を集中的に対策できる |
| **バックアップ / 復元** | 学習進捗・成績履歴を JSON ファイルに書き出し、別端末や再インストール時に復元できる |
| **オフライン対応** | Service Worker で全アセットをキャッシュ。ネットワーク不要で利用可能 |
| **PWA インストール案内** | 対応ブラウザでバナーが表示され、ホーム画面へ追加してアプリとして使える |
| **Apple デバイス対応** | apple-touch-icon / apple-mobile-web-app-capable 等のメタタグ設定済み |

## 技術スタック

- フロントエンド: Vanilla HTML/CSS/JS（フレームワークなし）
- ホスティング: GitHub Pages
- 外部依存: なし（ブラウザのみで動作）
- ローカル永続化: localStorage
